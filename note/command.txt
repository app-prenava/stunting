VALIDASI OUTPUT STEP 5B:
python - <<'PY'
import pandas as pd

df = pd.read_csv("Data/processed/05b_child_mother_stunting_cleaned.csv")

print(df.shape)
print("missing haz_score:", df["haz_score"].isna().sum())
print("missing is_stunted:", df["is_stunted"].isna().sum())

stunted = (df["is_stunted"] == 1).sum()
normal = (df["is_stunted"] == 0).sum()

print("stunted:", stunted)
print("normal:", normal)
print("prevalence:", stunted / len(df) * 100)
PY

VALIDASI FINAL (READY FOR MODELING):
python - <<'PY'
import pandas as pd

df = pd.read_csv("Data/processed/06_ml_ready_dataset.csv")

# 1. Tidak ada kolom ID
id_cols = ["household_id", "child_id", "mother_pidlink"]
print("ID columns present:", [c for c in id_cols if c in df.columns])

# 2. Tidak ada leakage
leak_cols = ["haz_score", "height_cm"]
print("Leakage columns present:", [c for c in leak_cols if c in df.columns])

# 3. Target valid
print("target unique:", df["is_stunted"].unique())

# 4. Semua numerik
non_numeric = df.select_dtypes(exclude=["int64", "float64"]).columns
print("non-numeric columns:", list(non_numeric))

# 5. Missing check
print("total missing:", df.isna().sum().sum())

print("\nDataset READY for Step 7 (Modeling)")
PY

CATATAN METODOLOGIS:

Pipeline ini terdiri dari:
- Step 1–4: Data integration & feature construction (Data Understanding + Data Preparation)
- Step 5: Target construction (label engineering)
- Step 6: ML preprocessing (feature engineering & encoding)

Output akhir Step 6:
Dataset numerik tanpa leakage, siap digunakan untuk supervised machine learning.

Distribusi target (~32% stunted):
- Tidak termasuk severe imbalance
- Tidak memerlukan SMOTE pada tahap awal
- Akan ditangani di modeling menggunakan:
  - class_weight
  - evaluation metrics (recall, F1-score)
